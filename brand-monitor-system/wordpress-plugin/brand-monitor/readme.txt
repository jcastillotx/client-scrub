Brand Monitor connects WordPress to the Brand Monitor backend to display real-time brand intelligence powered by Apify scraping and Claude AI.

== Description ==
* Dashboard with sentiment trends, alerts, and manual scrape triggers.
* Settings interface to manage API endpoints and keys.
* Automated notifications and email templates for crisis detection.

== Installation ==
1. Upload the `brand-monitor` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings â†’ Brand Monitor to configure API credentials.

== Frequently Asked Questions ==
= Do I need an API key? =
Yes, generate an API key inside the Brand Monitor backend and paste it into the plugin settings page.
